# JNTUK CGPA Calculator

A command-line Python application to calculate CGPA based on JNTUK grading regulations.

## Features
- Supports multiple semesters
- Handles decimal credit points
- Grade to point mapping as per JNTUK
- Detects failed subjects
- Simple CLI interface

## Grade Mapping
| Grade | Points |
|------|--------|
| S | 10 |
| A | 9 |
| B | 8 |
| C | 7 |
| D | 6 |
| E | 5 |
| F | 0 |

## How to Run
```bash
python cgpa_calculator.py
